# Startup Roast AI 🔥

> A free, browser-based startup idea evaluation tool. No backend, no APIs, no sign-up.

## Live Features

- 🎨 **Brand Score** – Analyzes startup name length, clarity, and keyword strength
- 📈 **Market Potential** – Evaluates audience specificity and market signals
- 💰 **Revenue Model** – Classifies and scores monetization strategy
- ⚔️ **Competition Risk** – Detects High / Medium / Low competition
- 🏆 **Overall Startup Score** – Weighted composite score
- 💡 **3 Strengths, 3 Weaknesses, 3 Actionable Suggestions**

## Tech Stack

- **React 18** + **Vite 5**
- **Tailwind CSS 3**
- 100% client-side deterministic scoring logic

## Getting Started

```bash
npm install
npm run dev
```

## Deploy to Vercel

```bash
npm run build
# Then drag the dist/ folder to vercel.com, or connect your GitHub repo
```
